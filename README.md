# Analyzing Geomagnetic Storms

This repository contains a project report on the relationship between solar wind disturbances and Earth’s geomagnetic response.

## Overview

The study focuses on the March 2015 St. Patrick’s Day geomagnetic storm using OMNIWeb data. It investigates:
- solar wind speed and density
- IMF Bz and magnetic reconnection
- dynamic pressure and solar wind electric field
- propagation delay from L1 to Earth
- Dst index evolution
- Burton model reconstruction
- storm probability mapping
- machine learning prediction of geomagnetic activity

## Contents

- `main.pdf` — final project report

## Methods Used

- Data cleaning and outlier removal
- Linear interpolation of missing values
- Cross-correlation for time delay estimation
- Power spectral density analysis
- Burton equation fitting
- Heatmap-based storm risk analysis
- XGBoost regression for Dst prediction

## Data Sources

- NASA OMNIWeb / Space Physics Data Facility (SPDF)

## Purpose

This report is intended as a study and analysis project on geomagnetic storm dynamics and forecasting.
